A simple C program that prints all even and odd numbers from a 2D array and counts their occurrences.
# Even and Odd Count in 2D Array (C Program)

## 📌 Description
This program prints all even and odd numbers from a 3×3 matrix and counts how many even and odd numbers are present.

## 🚀 How It Works
- Traverse the matrix using nested loops.
- Check each element using the modulus (%) operator.
- If divisible by 2 → Even.
- Otherwise → Odd.
- Count and display the total even and odd numbers.

## 📥 Input Matrix

```text
12 45 7
89 23 56
11 90 34
```

## 📤 Output

```text
Even = 12
Odd = 45
Odd = 7
Odd = 89
Odd = 23
Even = 56
Odd = 11
Even = 90
Even = 34

Even Count = 4
Odd Count = 5
```

## 🛠️ Concepts Used
- 2D Arrays
- Nested Loops
- If-Else Statements
- Modulus Operator (%)

## 🎯 Learning Outcome
- Matrix traversal
- Even/Odd checking
- Counting elements
- Working with nested loops
